GOALS

[G2] - The user can place default preferences
[G3] - The user has to be allowed to edit the location and the time of each scheduled appointment or cancel them
[G4] - The user has to be allowed to change the proposed travel by editing preferences
[G5] - The user has to recieve notifications about changes of the schedule (ex. cancellations of public transports, excessive traffic)  and change schedule accordingly
[G6] - When a ticket  is successfully booked or bought, the user is sent the ticket or the prenotation
[G8] - The application has to keep track of booked tickets
